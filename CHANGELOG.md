## [0.9.0] - February 25, 2026

### Added

- **Spec 02: MCP & Tools**
  - ToolRegistry: Centralized tool registration and discovery system
  - MCPToolAdapter: Model Context Protocol integration adapter
  - DynamicToolWatcher: Runtime tool monitoring and hot-reload support

- **Spec 06: Swarm Protocol**
  - Auction system for distributed task allocation
  - Guild system for collaborative agent groupings
  - Reputation system for agent trust scoring

- **Spec 09: Deep Observability**
  - Tracing: Distributed request tracing across services
  - Metrics: Performance and usage metrics collection
  - Session Replay: User session recording and playback
  - Flight Recorder: High-fidelity event capture for debugging

- **Spec 12: UX Strategy**
  - WhyButton: Explainable AI reasoning interface
  - Dream Log: Creative ideation and exploration journal
  - Monaco Diff: Advanced code diff visualization
  - FuzzyMenu: Intelligent command palette with fuzzy matching

- **Spec 14: Just-in-Time Tooling**
  - LibraryDiscoverer: Automatic library discovery and recommendations
  - CodeSynthesizer: AI-powered code generation from specifications
  - SkillCache: Intelligent caching of learned skills and patterns

- **Spec 18: Code Level Upgrades**
  - RAPTOR: Recursive Abstractive Processing for Tree-Organized Retrieval
  - Tool Critics: AI-powered tool evaluation and optimization
  - Vector Backends: Multiple vector storage backend support

- **Spec 19: Advanced Interaction**
  - Persona Engine: Dynamic agent persona management
  - Semantic Distiller: Intelligent content summarization
  - Contrastive Reasoning: AI reasoning through comparative analysis

- **Spec 20: SOTA Research Hub**
  - Declarative Signatures: Type-safe declarative function definitions
  - Artifacts: Rich content artifact management
  - Few-Shot Retriever: Intelligent example-based learning system

- **Spec 05: Metacognition**
  - Confidence Scoring: AI self-assessment of response quality
  - Streaming Auditor: Real-time streaming response validation

- **Web GUI**
  - 10 pages
  - 47 REST endpoints
  - 3 WebSocket channels

### Improved

- Test suite performance with markers and parallel execution
- Logging and warning cleanup
- Pytest configuration
