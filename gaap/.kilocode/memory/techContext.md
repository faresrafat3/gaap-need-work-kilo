# Technical Context: GAAP (Generative Autonomous Agent Platform)

## Core Stack
- **Language:** Python 3.12
- **Web Framework:** FastAPI (for API), Streamlit/Custom Web (for Dashboard)
- **AI Integration:** Multi-provider support (Groq, DeepSeek, G4F, WebChat Bridges)
- **Architecture:** Layered Autonomous Architecture (L0-L3)

## Key Components
- **GAAP Engine:** Core orchestrator.
- **Meta-Learning:** Experience analyzer and pattern extractor.
- **Self-Healing:** Healer module for automated error recovery.
- **Hierarchical Memory:** Advanced memory management for agents.
- **Validators:** Multi-stage validation (Security, Syntax, Compliance).
